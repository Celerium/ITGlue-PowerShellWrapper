# Get-ITGlueBaseURI

## SYNOPSIS
Shows the ITGlue base URI

## SYNTAX

```powershell
Get-ITGlueBaseURI [<CommonParameters>]
```

## DESCRIPTION
The Get-ITGlueBaseURI cmdlet shows the ITGlue base URI from
the global variable

## EXAMPLES

### EXAMPLE 1
```powershell
Get-ITGlueBaseURI
```

Shows the ITGlue base URI value defined in the global variable
