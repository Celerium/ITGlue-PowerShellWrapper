# Remove-ITGlueAPIKey

## SYNOPSIS
Removes the ITGlue API key

## SYNTAX

```powershell
Remove-ITGlueAPIKey [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Remove-ITGlueAPIKey cmdlet removes the ITGlue API key from
global variable

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-ITGlueAPIKey
```

Removes the ITGlue API key global variable
